# Changelog

All notable changes to PalServer-Manager will be documented in this file.

## [1.0.0] - 2026-07-22

### Features

- Initial release of PalServer-Manager desktop application.
- Multi-instance management for Palworld dedicated servers.
- Process lifecycle control (start, stop, force-kill, RAM trim).
- Graphical and raw text editor for `PalWorldSettings.ini`.
- Real-time monitoring dashboard (CPU/RAM, server FPS, player counts).
- Built-in file manager, player moderation database, and RCON terminal tab.
- Automated SteamCMD installation and template management.
